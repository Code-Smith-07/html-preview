## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required!

### Usage

1. **Clone or Download** this repository:
   ```bash
   git clone https://github.com/Code-Smith-07/html-preview.git
   cd html-preview
   ```

2. **Open the Editor**:
   - Simply open `Apps.html` in your web browser
   - Or use a local server for the best experience:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js
     npx http-server
     ```

3. **Start Coding**:
   - Write HTML in the HTML editor
   - Add CSS styles in the CSS editor
   - Add JavaScript functionality in the JS editor
   - Watch your creation come to life in the preview window!

4. **Save Your Work**:
   - Enter a name for your project in the "Save Your Zip file as" field
   - Click "Download as ZIP" to export your project
   - Extract the ZIP to get your HTML, CSS, and JS files

## ğŸ“– How It Works

### File Linking
When you download your project as a ZIP file, the files are named based on your input. For example, if you name your project "MyWebsite", you'll get:
- `MyWebsite.html`
- `MyWebsite.css`
- `MyWebsite.js`

**Important**: Make sure to link your CSS and JavaScript files correctly in your HTML:

```html
<!-- Link CSS -->
<link rel="stylesheet" href="MyWebsite.css">

<!-- Link JavaScript -->
<script src="MyWebsite.js"></script>
```
